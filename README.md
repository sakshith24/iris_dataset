#introduction 
This project implements a machine learning classification model using the Iris dataset to predict the species of an iris flower based on its sepal and petal measurements.
The dataset contains 150 samples with 4 features:
Sepal Length
Sepal Width
Petal Length
Petal Width

Target classes:
Iris-setosa
Iris-versicolor
Iris-virginica

i had applied logistic regression in which i got the accuracy of 94%
